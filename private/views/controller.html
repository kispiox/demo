Your First Controller

Create the folder structure for you application by running # mkdir -p src/NameSpace/Application. Create a subdirectory within this new structure called Controllers ( # mkdir src/NameSpace/Application/Controllers). This is the directory that will house all your controllers.

Now, create a new, empty controller.

src/NameSpace/Application/Controllers/DefaultController.php

<?php

namespace NameSpace\Application\Controllers;

use Kispiox\Controller;

class DefaultController extends Controller
{
}
