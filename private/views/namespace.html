Namespace Setup

Pick a namespace for your application. You can choose from common conventions such as VendorName\AppName, or domain notation such as com\example\app_name.

Once you've chosen a namespace, you'll need to update composer.json.

composer.json

{
    "name": "vendor/package",
    "description": "A description of you application",
    "authors": [
        {
            "name": "Your Name",
            "email": "your@email.ca"
        }
    ]
    "autoload": {
        "psr-4": {
            "NameSpace\\Application\\": "src/NameSpace/Application",
            "Kispiox\\Welcome\\": "src/Kispiox/Welcome/"
        }
    },
    "require": {
        "kispiox/kispiox": "dev-master"
    },
    "require-dev": {
        "phpunit/phpunit": "^5.7"
    },
    "license": "BSD-2-Clause",
}

Then you'll need to run # composer update to update the composer autoloader with the new information.
